class luchador.agent.misc.WienerNoise(shape, delta, dt, seed=None)¶

Bases: object

Generate noise with Wiener process, i.e. gaussian noise

Parameters:

shape : tuple

Shape of the noise to generate

delta, dt : float

Parameters to controll the process. dt represents the time difference between calls to sample() delta represents standard deviation

\[x \sim \mathcal{N} (\mu=0, \sigma=delta^2 * dt)\]

seed

Seed for random generator
reset()¶

Reset internal state. Do nothing for WienerNoise

sample()¶

Sample noise

See WienerNoise() constructor for the value sampled.

class luchador.agent.misc.OUNoise(shape, mu, sigma, theta, seed=None)¶

Bases: luchador.agent.misc.WienerNoise

Generate temporally correlated noise with Ornstein-Uhlenbeck process

Parameters:

shape : tuple

Shape of the noise to generate

mu, sigma, theta : float

Parameters to controll the process.

\[ \begin{align}\begin{aligned}x &\leftarrow \mu\\dx &\sim \theta * (\mu - x) + \sigma * \mathcal{N} (\mu=0, \sigma=1)\\x &\leftarrow x + dx\end{aligned}\end{align} \]

seed

Seed for random generator
reset()¶

Reset the internal state to \(mu\)

sample()¶

Sample noise

See OUNoise() constructor for the value sampled.
