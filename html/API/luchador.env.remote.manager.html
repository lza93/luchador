luchador.env.remote.manager module¶

Module for running server which manage remote environments

Manager Server is for starting new Environment server on local machine via POST request. Manager server can be created using create_manager_app() as following.

>>> import luchador.env
>>> app = luchador.env.remote.create_manager_app(env)
>>> server = luchador.env.remote.create_server(app)
>>> server.start()

Manager API¶

Start new environment server
URL /create
Method POST
Data Params environment: Environment configuration in JSON format port: integer
Success Response

Code: 200

Content (JSON)
environment: "Environment representation"
