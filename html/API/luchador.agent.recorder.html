Create state matrix from observations.

Parameters:

index : int

Index for the last observation to include in the resulting state.

For performance reason, argument validity check is omitted, but arguments must satisfy the following condition:

0 <= length - 1 <= index < len(self.actions)

length : int

The number of observations included in the state
Returns:

NumPy Array

Where the length of the first axis equals to the given length, and the other axis corresponds to the shape observation.
