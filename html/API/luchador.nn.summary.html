class luchador.nn.summary.SummaryWriter(output_dir)¶

Bases: object

Wrap tf.SummaryWriter for better flwexibility

Unlike Tensorflow’s native SummaryWrite, this SummaryWriter accepts NumPy Arrays for summarization. Generation of summary protocol buffer is handled internally.
