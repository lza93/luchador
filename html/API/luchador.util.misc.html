luchador.util.misc.get_subclasses(class_)¶

Get the list of all subclasses

This function is intended to use in get_* functions such as get_agent, get_env, get_layer, get_optimizer, get_initializer etc. for flexible access.

Parameters:

class_ : type

Class of which subclasses are searched
Returns:

list

List of subclasses of the given class
