luchador.configure module¶

Module for configuring luchador library

luchador.configure.get_nn_backend()¶

Get NN module backend

Returns:

str

Either theano or tensorflow
luchador.configure.get_nn_conv_format()¶

Get convolution data format of Tensorflow backend

Returns:

str

Either NCHW or NHCW. Refer to tf.nn.conv2d of the official Tensorflow documentation.
luchador.configure.get_nn_dtype()¶

Get default dtype of Tensorflow backend

Returns:

str

The name of default tf.DType
luchador.configure.set_nn_backend(backend)¶

Set NN module backend

Note

Backend must be set before luchador.nn module is imported.

Parameters:

backend : str

Either theano or tensorflow
luchador.configure.set_nn_conv_format(format_)¶

Set convolution data format of Tensorflow

Note

Convolution format has affect only in Tensorflow backend.

Parameters:

format_ : str

Either NHCW or NCHW. See``tf.nn.conv2d`` for the detail.
luchador.configure.set_nn_dtype(dtype)¶

Set default dtype for creating variables in Tensorflow backend

Note

This dtype has affect only in Tensorflow backend. To set default dtype of Theano backend, use floatX.

Parameters:

dtype : str

String expression of tf.DType.
