class luchador.nn.base.initializer.BaseXavier(uniform=True, seed=None, dtype=None)¶

Bases: luchador.nn.base.initializer.BaseInitializer

Implement Xavier initialization [R7] in Tensorflow manner [R8]

Parameters:

uniform : Bool

If True, uniform distribution is used. Otherwise normal distribution is used. See Notes.

seed : int or None

Seed value for random generator

dtype : str or None

Data type to sample. If None, default dtype is used.

Notes

The implementation of Xavier in Theano follows that of Tensorflow.[R8]_ The underlying distribution is computed in the following way.

when uniform=True,

\[\begin{split}a &= \sqrt{\frac{6}{fan_{in}+fan_{out}}}\\ W &\sim U[-a, a]\end{split}\]

When uniform=False,

\[\begin{split}\sigma &= \sqrt{\frac{3}{fan_{in}+fan_{out}}}\\ W &\sim N(0, \sigma)\end{split}\]

References

[R7](1, 2) Xavier Glorot and Yoshua Bengio (2010): Understanding the difficulty of training deep feedforward neural networks. International conference on artificial intelligence and statistics.
[R8](1, 2) https://www.tensorflow.org/versions/r0.11/api_docs/python/contrib.layers.html#xavier_initializer
