class luchador.util.mixin.StoreMixin¶

Bases: object

Provide common pattern to validate and store constructor arguments.

StoreMixin implements the follwoing methods.

_store_args(**args)¶

Store the given arguments to args attribute after validation

_validate_args(**args)¶

Validate the given arguments

Subclassses inheriting this mixin can add custom validation scheme by overriding this method.

See also

CompareMixin
Base mixin which adds equality comparison to StoreMixin
SerializeMixin
Subclass mixin which serialize object with arguments

Examples

>>> class Foo(StoreMixin):
>>>     def __init__(self, number, string):
>>>         self._store_args(numer=number, string=string)
>>>
>>>     def _validate_args(number, string):
>>>         if not isinstance(number, float):
>>>             raise TypeError('Argument `number` must be float type.')
>>>         if not isinstance(string, str):
>>>             raise TypeError('Argument `string` must be string type.')
>>>
>>> foo = Foo(0.5, 'bar')
>>> foo.args['number'] == 0.5
True
>>> foo.args['string'] == 'bar'
True
>>> Foo('bar', 1)
TypeError: Argument `number` must be float type.
class luchador.util.mixin.CompareMixin¶

Bases: luchador.util.mixin.StoreMixin

Extend StoreMixin by providing method to compare args

__eq__(other)¶

Checks if the other object has same configuration with this object

Same configuration here means objects are of the same (or subclassed) type and have the same args attribute value.

__ne__(other)¶

Checks if the other object is not equal to this object

See also

StoreMixin
Base mixin which stores costructor arguments
SerializeMixin
Subclass mixin which serialize object with arguments

Examples

>>> class Foo(CompareMixin):
>>>     def __init__(self, arg):
>>>         self._store_args(arg=arg)
>>>
>>> foo1 = Foo('bar')
>>> foo2 = Foo('bar')
>>> foo3 = Foo('foo')
>>> foo1 == foo2
True
>>> foo1 == foo3
False
